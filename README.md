# derckprinzhorn.com

Personal website for Derck Prinzhorn - AI Researcher & Consultant.

## Structure

```
├── index.html          # Main single-page site
├── css/
│   └── styles.css      # All styles (light/dark mode)
├── js/
│   └── main.js         # Theme toggle, animations, mobile menu
├── images/
│   ├── profile-derck-circle.png
│   └── publications/   # Research paper thumbnails
├── pdfs/
│   └── cv/             # CV documents
├── favicon.ico
└── CNAME               # Custom domain config
```

## Features

- Responsive single-page design
- Light/dark mode with system preference detection
- Smooth scroll navigation
- Scroll reveal animations
- Mobile-friendly navigation

## Development

Simply open `index.html` in a browser or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Deployment

Deployed automatically via GitHub Pages to [derckprinzhorn.com](https://derckprinzhorn.com)
